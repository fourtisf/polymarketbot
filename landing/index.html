How it works

Three rules, run every five minutes.

The bot wakes up at the start of each 5-minute window, evaluates the edge, and either takes a position or skips.

01 · OBSERVE

Read the tape

Pull BTC mid from Binance + Polymarket UP/DOWN ask prices. Compute delta vs the price-to-beat at window expiry.

02 · DECIDE

Score the edge

Combine momentum, microstructure imbalance, and historical hit-rate into a confidence score from 0–100.

03 · EXECUTE

Sign and submit

If confidence clears the bar, size with Kelly-style risk caps and submit a signed CLOB order. Otherwise skip.
